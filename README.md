## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作成します。
2. このフォルダの中身をリポジトリ直下にアップロードします。
3. GitHubの `Settings` → `Pages` で、公開元を `main` ブランチの `/root` にします。
4. 表示されたURLをレビュアーに共有します。
